createBanner()method 
public function createBanner(adSize:int, adPosition:int, bannerId:String = null, adMobId:String = null, autoShow:Boolean = false):void

Create the adMob Banner in relative position

Parameters

adSize:int — Banner AdmobSize to be used
 
adPosition:int — Banner Position to be used
 
bannerId:String (default = null) — Unique Banner Id identifier, used for identifie the banner when multiple banner are use in the same application. (please note, this is not the adMobId). Optional, if an id is not given an automatic id is generated
 
adMobId:String (default = null) — Unique AdMobId for the banner. Optional, if an id is not given the common Id will be used
 
autoShow:Boolean (default = false) — Automatic visualization. Optional, Default = false If the autoShow is set to true the banner will be automatically render when is recieved, otherwise will need to be render manualy After BANNER_RECIEVED event.
